# Jonny's Branch Print Agent

Servicio Node.js saliente que reclama trabajos de una sola sucursal y los imprime en QZ Tray 2.2.6 mediante `wss://localhost:8181`. Consulte `../docs/branch-print-agent.md` para provisionamiento, operacion y diagnostico.

No copie tokens al repositorio. Cree `.env` desde `.env.example` solamente en la PC de la sucursal.
